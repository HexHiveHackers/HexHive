# Resource packs and reference asset repositories

Other places that host the kinds of assets HexHive aggregates. Useful as
upstream sources for your own listings, and as reference points for what
"good" coverage of a category looks like.

> ⚠️ Inclusion here is informational only. HexHive is not affiliated with these
> projects. Each one has its own licensing, attribution, and posting rules —
> read theirs before you redistribute on HexHive.

## Sprites

- **[PMD Collab — sprites & portraits](https://sprites.pmdcollab.org/)**
  Pokémon Mystery Dungeon community spritedex, with portraits and overworld
  sprites for every species, contributor-credited and version-tracked. The
  scope and the way they handle credits are worth studying as a coverage
  benchmark.

- **[Team Aqua's Hideout — Asset Repo](https://github.com/TeamAquasHideout/Team-Aquas-Asset-Repo)**
  Large GitHub-based asset repository covering trainer fronts/backs, overworld
  sprites, tilesets, items, UI, battle backgrounds, audio, and Pokémon
  Essentials packs. Used as the seed corpus for `scripts/seed-team-aqua.ts`.

## Adding to this list

Open a PR (or issue) with:

- The project name and a one-line description of what it covers.
- A link to the project (homepage or canonical repo).
- A note on licensing/attribution if it isn't obvious from the link.
